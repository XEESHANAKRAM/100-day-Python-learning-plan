Week 1: Introduction to Python (Days 1-7)

Day 1: Introduction to Python, Installing Python, Setting Up the Environment

  • Understand what Python is and its usefulness in DevOps.
  • Install Python from the official website.
  • Set up a Python development environment using a text editor or IDE.
  • Create a simple Python script to test the setup.
Example Code:
print("Hello, Python for DevOps!")
Exercise: Write a script that prints your name and a message about why you are learning Python.
# Example solution print("My name is [Your Name] and I'm learning Python for DevOps to automate tasks!")

Day 2: Basic Syntax, Variables, and Data Types

  • Learn the basic syntax and how Python code is structured.
  • Understand variables and different data types (integers, floats, strings, booleans).
  • Practice with a script to store and print various data types.
Example Code:
name = "DevOps Engineer"
age = 25
pi = 3.14159
is_learning = True
print(f"Name: {name}, Age: {age}, Pi: {pi}, Learning Python: {is_learning}")
Exercise: Create variables to store your favorite color, number, and a boolean value indicating if you like Python. Print these values.
# Example solution favorite_color = "Blue"
favorite_number = 7
likes_python = True
print(f"Favorite Color: {favorite_color}, Favorite Number: {favorite_number}, Likes Python: {likes_python}")

Day 3: String Manipulation and Formatting

  • Explore string operations like concatenation, slicing, and methods.
  • Learn different ways to format strings for output.
  • Write scripts using string formatting and manipulation.
Example Code:
name = " DevOps Engineer "
print(name.strip().upper())
print(name.replace("DevOps", "Cloud"))
greeting = "Hello, {}!".format("world")
f_greeting = f"Hello, {name.strip()}!"
print(greeting)
print(f_greeting)
Exercise: Ask the user for their name and age, and print a message using both the `format()` method and f-strings.
# Example solution name = input("Enter your name: ")
age = input("Enter your age: ")
print("Your name is {} and you are {} years old.".format(name, age))
print(f"Your name is {name} and you are {age} years old.")

Day 4: Basic Operators (Arithmetic, Comparison, Logical)

  • Understand and use arithmetic operators.
  • Learn about comparison and logical operators.
  • Write scripts that perform calculations and logical checks.
Example Code:
a = 10
b = 3
print(a + b, a - b, a * b, a / b, a ** b, a % b)
print(a > b, a == b, a != b)
is_python = True
is_java = False
print(is_python and is_java, is_python or is_java, not is_java)
Exercise: Write a script that takes two numbers from the user, compares them, and prints out which one is larger or if they are equal.
# Example solution num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))
if num1 > num2:
 print("The first number is larger.")
elif num1 < num2:
 print("The second number is larger.")
else:
 print("Both numbers are equal.")

Day 5: Input and Output Functions

  • Learn how to take input from the user using the input() function.
  • Understand how to display output using the print() function.
  • Write scripts that interact with the user and display results.
Example Code:
name = input("What is your name? ")
age = int(input("What is your age? "))
print(f"Hello {name}, you are {age} years old.")
Exercise: Write a script that asks for the user's name, age, and favorite number, then prints out a formatted message with this information.
# Example solution name = input("Enter your name: ")
age = input("Enter your age: ")
favorite_number = input("Enter your favorite number: ")
print(f"Hello {name}, you are {age} years old and your favorite number is {favorite_number}.")

Day 6: Lists, Tuples, and Sets

  • Learn how to create and use lists, tuples, and sets.
  • Practice adding, removing, and accessing elements in these data structures.
  • Write scripts to manipulate and process collections of data.
Example Code:
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)
print(fruits[1])

veggies = ("carrot", "broccoli", "spinach")
print(veggies)

colors = {"red", "blue", "green"}
colors.add("yellow")
print(colors)
Exercise: Create a list of your favorite foods, add a new food to the list, and then print the list. Do the same for a tuple of favorite drinks and a set of favorite sports.
# Example solution
favorite_foods = ["pizza", "sushi", "tacos"]
favorite_foods.append("pasta")
print(f"Favorite Foods: {favorite_foods}")

favorite_drinks = ("coffee", "tea", "juice")
print(f"Favorite Drinks: {favorite_drinks}")

favorite_sports = {"football", "basketball", "tennis"}
favorite_sports.add("volleyball")
print(f"Favorite Sports: {favorite_sports}")

Day 7: Practice Problems and Review

  • Review all topics covered during the week.
  • Solve practice problems to reinforce the concepts.
  • Identify challenging areas and revisit those topics for better understanding.
Exercise 1: Write a script that calculates the area of a rectangle. Take input from the user for width and height.
# Example solution
width = float(input("Enter the width of the rectangle: "))
height = float(input("Enter the height of the rectangle: "))
area = width * height
print(f"The area of the rectangle is {area}.")
Exercise 2: Create a list of numbers, then write a script to calculate the sum and average of the numbers in the list.
# Example solution
numbers = [10, 20, 30, 40, 50]
total = sum(numbers)
average = total / len(numbers)
print(f"Sum: {total}, Average: {average}")
